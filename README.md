# Notice Module

Simple MERN notice board module.

## Roles

- Principal: create, edit, read, delete
- Teacher: edit, read
- Student: read
- Admin: delete, read

## Structure

- `backend`: Express + MongoDB API
- `frontend`: Vite + React + Tailwind client

## Run

1. Start MongoDB Atlas and copy your connection string into `backend/.env`.
2. Run the backend and frontend in separate terminals.
